The typeCharts library is a meant to create an easier way for Typescript programmers to create var graphs.

using this library, all you have to do is specify just a few variables(can be seen in the examples) to display a bar graph on screen.


If you have any new features, bug fixes or improvements, feel free to fork and change this library to achieve many great things.


Thanks for your time,

Bar :)


P.S.

you have to compile the file which will be importing typeCharts.ts with the --amd module (tsc --module amd <path to file>).


This library also requires that you include Jquery and requireJS in the html doc.
